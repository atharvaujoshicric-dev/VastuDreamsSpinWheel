# 🎡 Wheel Of Fortune

A fun, interactive "Wheel of Fortune" application built with **Streamlit**, **SVG**, and **JavaScript**.

## 🚀 Features
- **Interactive Spin Wheel**: Smooth CSS-based animations.
- **Dynamic Prize Configuration**: Add, edit, or remove prizes in real-time via the Streamlit sidebar.
- **Confetti Celebration**: Visual feedback with a confetti explosion when you win!
- **SVG Graphics**: Scalable and crisp visuals.
- **Winner Detection**: Real-time winner calculation based on rotation.

## 🛠️ Tech Stack
- [Streamlit](https://streamlit.io/)
- HTML5 / CSS3 / JavaScript
- SVG

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/atharvaujoshicric-dev/Spin-Wheel.git
   cd Spin-Wheel
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   streamlit run app.py
   ```
